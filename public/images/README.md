# Site Images

Replace any file here with your own photos. **Keep the same filename** so the website picks them up automatically.

| File | Used on |
|------|---------|
| `hero-home.jpg` | Home page hero & footer CTA background |
| `about-team.jpg` | About page hero & story section |
| `contact.jpg` | Contact page hero & sidebar photo |
| `why-choose.jpg` | Home "Why Choose SRL" section |
| `logistics.jpg` | Logistics service page hero |
| `cloud.jpg` | Cloud hosting service page hero |
| `email.jpg` | Email & domains service page hero |
| `software.jpg` | Software development service page hero |
| `gallery-*.jpg` | Project gallery on Home & About pages |

## Tips

- Recommended size for hero images: **1920×1080 px** (JPG or WebP)
- Recommended size for gallery images: **800×600 px** or larger
- To change paths or add new photos, edit `src/data/images.js`
